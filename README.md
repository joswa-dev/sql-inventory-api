# SQL Inventory API

A simple backend project built with Python and SQLite to manage job listings.
This project demonstrates core SQL and backend development concepts.

## Features
- Create job records
- View all jobs
- Update job location
- Delete jobs
- SQLite database integration
- Clean modular code structure

## Tech Stack
- Python
- SQLite
- SQL

## Project Structure
- `db.py` – Database connection
- `models.py` – Table creation
- `crud.py` – CRUD operations
- `jobtrack.db` – SQLite database

## How to Run
```bash
python models.py
python crud.py
